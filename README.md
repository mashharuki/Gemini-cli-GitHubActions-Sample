# Gemini-cli-GitHubActions-Sample
Gemini CLI + GitHub Actionsのサンプル実装

## 事前セットアップ

- Gemini CLIのインストール
- GitHub リポジトリでGemini APIキーをSecretに登録
  - `GEMINI_API_KEY`
- CLIとGitHub Actionsの連携
  - `gemini /setup-github`
